# 夜莺钱包用户使用手册

本文面向普通用户，说明如何安全地安装、创建、导入和使用夜莺钱包浏览器插件。

夜莺钱包是一个浏览器扩展钱包，支持查看余额、收发资产、切换网络、连接 DApp、管理授权网站、联系人、备份同步和 MPC 门限钱包等功能。

在插件中，可以点击主页面右上角更多菜单，然后点击“帮助文档”打开本手册。

## 1. 使用前必读

请先理解以下安全原则：

- 钱包密码只用于保护本机插件数据。忘记密码后，钱包无法替你找回密码。
- 助记词和私钥可以完全控制你的资产。任何人拿到助记词或私钥，都可以转走资产。
- 不要把助记词或私钥发送到微信、邮箱、网盘、聊天工具或任何网站。
- 不要在不可信网站上连接钱包或签名。
- 第一次使用新地址、新网络或大额转账前，建议先做一笔小额测试。
- 转账一旦上链通常无法撤回，请务必核对网络、地址、资产和金额。

## 2. 安装插件

开发者模式安装步骤：

1. 将钱包代码下载到本地。
2. 在 Chrome 或兼容浏览器中打开 `chrome://extensions/`。
3. 打开右上角“开发者模式”。
4. 点击“加载未打包的扩展程序”。
5. 选择钱包项目目录。
6. 安装完成后，可以把夜莺钱包固定到浏览器工具栏。

如果插件已安装，浏览器工具栏会显示夜莺钱包图标。点击图标即可打开钱包。

## 3. 创建或导入钱包

首次打开插件时，可以选择“创建钱包”或“导入钱包”。

### 创建新钱包

适合第一次使用夜莺钱包的用户。

1. 点击“创建钱包”。
2. 输入钱包名称。
3. 选择钱包类型。普通用户建议使用 `HD Wallet`。
4. 设置至少 8 位的钱包密码。
5. 再次输入密码确认。
6. 点击“创建钱包”。
7. 创建完成后，立即备份助记词。

备份建议：

- 把助记词抄写到纸上，放在安全位置。
- 确认每个单词的拼写和顺序正确。
- 不要截图，不要复制到联网文档，不要发送给任何人。

### 导入钱包

如果你已有钱包，可以通过助记词或私钥导入。

1. 点击“导入钱包”。
2. 选择“助记词”或“私钥”。
3. 输入钱包名称。
4. 输入助记词或私钥。
5. 设置或输入本地钱包密码。
6. 点击“导入钱包”。

导入成功后，请确认显示的钱包地址与你预期一致。

## 4. 解锁和锁定钱包

钱包在锁定状态下不会直接显示资产和执行敏感操作。

- 解锁：点击插件，输入钱包密码，点击“解锁”。
- 锁定：在主页面右上角更多菜单中点击“锁定钱包”。
- 如果 DApp 请求连接或签名时钱包已锁定，钱包会先要求输入密码，然后继续当前授权流程。

建议在公用电脑、临时设备或离开电脑前主动锁定钱包。

## 5. 主页面功能

主页面主要包含以下区域：

- 当前账户：显示账户名称和地址，可进入账户管理。
- 当前网络：显示当前连接的区块链网络。
- 资产列表：显示原生资产和已添加通证余额。
- 活动：显示交易记录。
- 转账按钮：进入转账页面。
- 更多菜单：进入联系人、网站管理、网络管理、设置和锁定钱包。

点击账户地址或复制按钮可以复制收款地址。点击账户详情可以查看二维码并导出二维码。

## 6. 账户管理

在主页面点击账户区域，可以进入“账户管理”。

常用操作：

- 切换账户：选择列表中的账户。
- 创建账户：点击“创建钱包”或“添加账户”，输入账户名称。
- 导入钱包：点击“导入钱包”，输入助记词或私钥。
- 修改名称：进入账户详情后编辑账户名称。
- 查看地址二维码：进入账户详情查看或导出二维码。
- 查看助记词或私钥：需要输入密码后才能显示。
- 删除账户：删除前必须确认已备份助记词或私钥。

重要说明：

- 删除账户只删除本地插件中的记录，不会删除链上资产。
- 如果没有备份助记词或私钥，删除后可能无法再恢复该账户。
- 私钥导入的钱包通常不能像 HD 钱包一样继续派生子账户。

## 7. 网络管理

夜莺钱包支持多个 EVM 网络。当前内置网络包括：

- YeYing Mainnet，原生资产为 `YYT`。
- Ethereum Mainnet，原生资产为 `ETH`。

切换网络：

1. 在主页面或转账页面打开网络选择器。
2. 选择目标网络。
3. 等待余额刷新。

添加自定义网络：

1. 打开右上角更多菜单。
2. 点击“网络管理”。
3. 点击“新增网络”。
4. 填写网络名称、RPC URL、Chain ID、货币符号和区块浏览器地址。
5. 点击“添加网络”。

请只添加可信 RPC 和可信网络。恶意或错误的 RPC 可能显示错误余额、错误交易信息，甚至诱导你签署危险交易。

## 8. 通证管理

钱包会显示原生资产，也支持 ERC-20 通证。

Ethereum Mainnet 已内置：

- `USDC`
- `USDT`

添加其他通证：

1. 在资产列表中点击“添加通证”。
2. 选择通证所在网络。
3. 输入通证合约地址。
4. 填写通证符号、小数位和名称。
5. 点击“添加通证”。

添加通证前请确认合约地址来自可信来源。任何人都可以创建同名通证，名称和符号不能作为安全依据。

## 9. 收款

收款步骤：

1. 确认当前账户地址。
2. 复制地址，或进入账户详情展示二维码。
3. 告诉对方正确的网络和资产。
4. 等待链上确认后，在余额或活动记录中查看结果。

注意事项：

- EVM 网络地址格式通常都是 `0x...`，但不同网络的资产不能混发。
- 例如，从 Ethereum Mainnet 发送 ETH、USDC 或 USDT 时，应确认对方使用的是以太坊网络。
- 如果从交易所提现，请选择与钱包当前收款网络一致的链。

## 10. 转账

转账步骤：

1. 点击主页面“转账”。
2. 选择网络。
3. 选择要发送的资产。
4. 输入接收地址，或从联系人中选择。
5. 输入金额。
6. 查看预计手续费。
7. 点击“发送”。
8. 再次确认地址、资产、金额和网络。
9. 提交后在“活动”中查看交易状态。

手续费说明：

- 发送原生资产时，手续费从同一种原生资产中扣除。例如以太坊网络发送 ETH 时，手续费也用 ETH 支付。
- 发送 ERC-20 通证时，也需要原生资产支付手续费。例如以太坊网络发送 USDC 或 USDT 时，需要账户中有 ETH 支付 gas。
- 手续费预估失败通常表示交易当前无法正常执行。常见原因包括余额不足、通证余额不足、RPC 不可用、合约拒绝交易或网络状态异常。

常见转账提示：

| 提示 | 含义 | 处理方式 |
| --- | --- | --- |
| `余额不足` | 原生资产不足，无法支付转账金额或手续费 | 减少金额，或先充值原生资产 |
| `通证余额不足` | 要发送的 ERC-20 通证余额不足 | 检查通证余额和小数位 |
| `RPC 不可用` | 当前网络节点无法响应 | 切换网络 RPC，稍后重试，或检查网络连接 |
| `交易会失败: ...` | 合约预估时返回回滚原因 | 根据提示检查合约、授权、余额或收款地址 |
| `预估失败: ...` | 钱包无法明确归类错误 | 检查网络、余额、地址和 RPC |

## 11. 连接 DApp 和签名

当网站请求连接钱包时，钱包会弹出授权页面。

连接请求通常表示网站希望读取你的账户地址和余额。连接前请检查：

- 网站域名是否正确。
- 是否是你主动打开的网站。
- 是否真的需要连接钱包。

签名请求比连接请求更敏感。签名前请检查：

- 签名内容是否可读。
- 是否是登录签名、授权签名还是交易签名。
- 是否包含你不理解的权限、时间范围或资源描述。
- 是否来自可信网站。

如果不确定，点击“拒绝”。拒绝连接或签名不会影响钱包中的资产。

### 添加网络和添加通证请求

DApp 可能请求钱包添加网络或添加通证。确认前请检查：

- 网络名称、RPC URL、Chain ID 和区块浏览器是否可信。
- 通证合约地址是否正确。
- 网站是否可信。

如果信息来源不可靠，不要添加。

## 12. 网站管理

打开右上角更多菜单，点击“网站管理”，可以查看已授权的网站。

你可以：

- 搜索授权网站。
- 查看授权详情。
- 撤销单个网站授权。
- 清除所有授权。

建议定期清理不再使用或不信任的网站授权。

## 13. 联系人

联系人用于保存常用地址，降低手动输入地址出错的风险。

添加联系人：

1. 打开更多菜单。
2. 点击“联系人”。
3. 点击“添加联系人”。
4. 输入名称、地址和备注。
5. 保存。

联系人支持导入和导出 JSON，便于在设备之间迁移。

注意：联系人只是本地通讯录。转账前仍需确认地址和网络正确。

## 14. 备份与同步

备份与同步用于在同一助记词的钱包设备之间同步部分本地数据。

当前同步对象包括：

- 账户名称。
- 账户数量。
- 联系人。
- 网络配置。
- 更新时间戳。

不会同步：

- 私钥。
- 助记词。
- 硬件钱包账户。
- 私钥导入账户。
- 交易记录。
- 授权站点。
- 代币余额。

启用方式：

1. 打开“设置”。
2. 找到“备份与同步”。
3. 输入 WebDAV 地址。
4. 选择认证方式：SIWE、UCAN 或 Basic。
5. 完成登录或保存凭证。
6. 开启“启用备份与同步”。
7. 点击“立即同步”测试。

如果出现同步冲突，页面会显示冲突项。请根据实际情况选择使用本地数据或远端数据。

更多细节见 [WebDAV 备份同步说明](./WebDAV备份同步说明.md)。

## 15. 安全设置

在“设置”页面可以进行安全操作。

### 修改密码

1. 打开“设置”。
2. 点击“修改密码”。
3. 输入旧密码。
4. 输入新密码并确认。
5. 点击“修改”。

修改密码只改变本机插件的加密密码，不会改变链上账户、地址、助记词或私钥。

### 重置钱包

重置钱包会删除本地插件数据。

重置前请确认：

- 已备份助记词或私钥。
- 已导出必要联系人或网络配置。
- 知道重置后需要重新导入钱包。

如果没有备份密钥，重置可能导致永久无法找回资产。

## 16. MPC 门限钱包

MPC 门限钱包适合多成员共同管理资产的场景。普通用户如果只是个人使用，可以先忽略此功能。

创建 MPC 钱包前，请确认：

- 已理解门限签名的参与者和阈值。
- 已准备好参与者联系人。
- 已配置协调器地址和 UCAN 授权。
- 所有参与者都能加入会话并完成协作。

MPC 相关功能包括创建会话、加入会话、发送消息、查看会话列表和日志审计。

更多设计细节见 [MPC 门限钱包方案](./MPC门限钱包方案.md)。

## 17. 常见问题

### 忘记钱包密码怎么办？

钱包无法找回密码。如果你已备份助记词或私钥，可以重置钱包后重新导入。如果没有备份，可能无法恢复账户。

### 为什么余额没有显示？

可能原因：

- 当前网络选择错误。
- RPC 节点不可用。
- 地址没有该网络资产。
- 资产是通证，但尚未添加到钱包。

可以尝试切换网络、刷新、检查 RPC，或手动添加通证。

### 为什么发送 USDC 或 USDT 还需要 ETH？

USDC 和 USDT 是以太坊上的 ERC-20 通证。发送这些通证时，交易仍发生在 Ethereum Mainnet 上，所以需要 ETH 支付手续费。

### 为什么手续费显示“RPC 不可用”？

钱包无法从当前 RPC 节点获取预估结果。可以检查网络连接，稍后重试，或在网络管理中更换可信 RPC。

### 为什么 DApp 没有弹出授权窗口？

可以尝试：

- 确认钱包插件已启用。
- 确认当前网站支持浏览器扩展钱包。
- 刷新网页后重试。
- 点击插件图标查看是否需要先解锁。
- 检查是否已在网站管理中撤销或清除授权。

### 为什么签名页面显示很多内容？

签名内容来自 DApp。钱包会尽量展示可读信息，例如 SIWE 登录信息、授权能力和交易详情。请确认内容可信再签名。

### 删除账户后资产会消失吗？

删除本地账户不会删除链上资产。但如果没有备份助记词或私钥，你可能无法再次控制这些资产。

### 可以把同一个地址用于多个网络吗？

EVM 网络通常使用相同地址格式，但资产存在于不同网络。收款和转账时必须确认网络一致。

## 18. 使用检查清单

创建或导入后：

- 已备份助记词或私钥。
- 已确认地址正确。
- 已设置可靠密码。
- 已了解如何锁定钱包。

收款前：

- 已确认当前账户地址。
- 已告诉对方正确网络。
- 大额收款前已做小额测试。

转账前：

- 已确认网络。
- 已确认资产。
- 已确认接收地址。
- 已确认金额。
- 已确认手续费。
- 发送通证时，账户中有足够原生资产支付手续费。

连接 DApp 前：

- 已确认网站域名。
- 已理解连接或签名请求。
- 不确定时选择拒绝。
